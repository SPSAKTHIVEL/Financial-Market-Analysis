# 📊 US Stock Market Analysis

This project analyzes historical US stock market data from major exchanges to understand market trends and stock behavior using data science techniques.

## 🚀 Project Overview
The project performs exploratory data analysis on stock price data including open, high, low, close prices, adjusted close, and trading volume. It also calculates financial indicators to study trends and volatility.

## 📂 Dataset
The dataset contains historical stock data for multiple US companies with the following features:

- Date
- Open
- High
- Low
- Close
- Adjusted Close
- Volume

## 🔎 Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Stock price trend visualization
- Moving average analysis
- Daily return calculation
- Volatility analysis

## 🛠️ Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
